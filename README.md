# Portfolio Website

A personal portfolio website showcasing web applications and tools.

## Structure

- `apps/` - Collection of web applications and tools
- `assets/` - Static assets including images, icons, and favicons
- `index.html` - Main portfolio homepage
- `apps/index.html` - Apps showcase page

## Features

- Responsive design
- Modern UI with glassmorphism effects
- Neural network background animation
- SEO optimized
- Accessible navigation

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome Icons
- Google Fonts (Inter)

## SEO Files

- `sitemap.xml` - XML sitemap for search engines
- `robots.txt` - Robots exclusion protocol
- `llms.txt` - Information for AI/LLM crawlers

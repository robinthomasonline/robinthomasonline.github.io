# Habit Tracker

Track daily habits with streak counters and calendar views.

## Features

- Daily habit tracking
- Recurring habits support
- Streak counter
- Calendar view
- Statistics dashboard
- Local storage persistence
- Habit completion history

## Files

- `index.html` - Main application file
- `habit.css` - Stylesheet
- `habit.js` - Application logic

## Usage

Open `index.html` in a web browser. Add habits and mark them as complete each day.

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- Local Storage API

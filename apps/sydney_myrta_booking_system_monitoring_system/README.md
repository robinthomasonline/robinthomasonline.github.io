# MyRTA Booking System Monitor

Free browser extension that monitors MyRTA booking availability changes and alerts you when new test dates become available.

## Features

- Monitor MyRTA booking page for visual changes
- Instant sound and desktop notifications
- Automatic page refresh every 10 minutes
- Background monitoring while you work
- Easy step-by-step installation guide
- Works with Chrome, Edge, and Chromium browsers

## Files

- `index.html` - Installation and usage guide
- `sydney_myrta_booking_system_monitoring_system.zip` - Extension package

## Installation

1. Download the extension ZIP file
2. Extract to a folder
3. Enable Developer Mode in your browser
4. Load the unpacked extension
5. Follow the step-by-step guide in `index.html`

## Usage

1. Book a test with any available date
2. Navigate to the Change Date page
3. Configure extension settings
4. Start monitoring
5. Receive alerts when new dates become available

## Technologies

- Browser Extension (Chrome/Edge)
- JavaScript
- Chrome Extension APIs

## Disclaimer

This extension only monitors visual changes on the MyRTA booking page. It does not create new booking slots or have any connection to Service NSW or government organizations.

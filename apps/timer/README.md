# Work Timer

A productivity timer application with Pomodoro technique support for tracking work sessions and tasks.

## Features

- Stopwatch and countdown modes
- Pomodoro timer (25 min work, 5 min break)
- Task management
- Quick preset timers
- Daily statistics
- CSV export functionality

## Files

- `index.html` - Main application file
- `timer.css` - Stylesheet
- `timer.js` - Application logic
- `beep.mp3` - Sound notification
- `stopwatch.png` - App icon

## Usage

Open `index.html` in a web browser.

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- Web Audio API
- Local Storage API

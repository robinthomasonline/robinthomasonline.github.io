# JSON Formatter

Format, validate, minify, and beautify JSON instantly.

## Features

- Format and beautify JSON
- Minify JSON
- Validate JSON syntax
- Copy to clipboard
- Customizable indent size
- File size and compression stats
- Error highlighting

## Files

- `index.html` - Main application file
- `json.css` - Stylesheet
- `json.js` - Application logic

## Usage

Paste JSON text or upload a JSON file. Format, validate, or minify as needed.

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- Clipboard API
- File API

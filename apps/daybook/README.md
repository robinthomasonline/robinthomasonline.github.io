# Daybook

Track income and expenses for small shops with customer details and invoice generation.

## Features

- Income and expense tracking
- Customer name and details
- Invoice generation
- CSV and JSON export
- Shop settings management
- Edit and delete entries
- Local storage persistence
- Print-ready invoices

## Files

- `index.html` - Main application file
- `daybook.css` - Stylesheet
- `daybook.js` - Application logic

## Usage

Open `index.html` in a web browser. Add income and expense entries with customer details.

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- Local Storage API
- Print API

# Password Generator

Generate secure, compliant passwords with customizable settings and compliance presets.

## Features

- Customizable password length
- Character type selection (uppercase, lowercase, numbers, symbols)
- Compliance presets:
  - ISO 27001
  - PCI DSS
  - CSA
  - HIPAA
  - NIST
  - GDPR
- One-click copy to clipboard
- Password strength indicator
- Security best practices

## Files

- `index.html` - Main application file
- `password.css` - Stylesheet
- `password.js` - Application logic

## Usage

Open `index.html` in a web browser. Select your requirements and generate secure passwords.

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- Clipboard API

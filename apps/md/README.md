# Markdown Converter

Convert Markdown files to DOC and PDF formats instantly with live preview.

## Features

- Markdown to DOC conversion
- Markdown to PDF conversion
- Live preview
- File upload support
- Word count display
- Syntax highlighting
- Customizable margins

## Files

- `index.html` - Main application file
- `converter.css` - Stylesheet
- `converter.js` - Application logic

## Usage

Paste Markdown text or upload a Markdown file, then export to DOC or PDF format.

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- Marked.js (Markdown parser)
- html2canvas (for PDF generation)
- File API

# Kanban Board

A minimalist drag-and-drop task management board with three-column workflow.

## Features

- Drag-and-drop task management
- Three-column workflow (To Do, In Progress, Done)
- Task editing and deletion
- Priority levels
- User assignment
- CSV export and import
- Local storage persistence

## Files

- `index.html` - Main application file
- `kanban.css` - Stylesheet
- `kanban.js` - Application logic

## Usage

Open `index.html` in a web browser. Drag tasks between columns to update their status.

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- HTML5 Drag and Drop API
- Local Storage API

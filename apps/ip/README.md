# IP Address Locator

Get comprehensive geographical and network information for any IP address.

## Features

- Real-time IP geolocation
- Complete IPAPI data display
- Interactive location map
- Network and ISP information
- Country and timezone details
- IPv4 and IPv6 support

## Files

- `index.html` - Main application file
- `ip.css` - Stylesheet
- `ip.js` - Application logic

## Usage

Enter an IP address or use your current IP. The app fetches data from IPAPI service.

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- IPAPI integration
- Leaflet.js (for maps)

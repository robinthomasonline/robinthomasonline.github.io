# Co-Founder Pie

Calculate fair equity distribution for startup co-founders based on contributions, tasks, and responsibilities.

## Features

- Equity distribution calculator
- Support for 2-10 founders
- Weighted task scoring
- Automatic percentage calculations
- Local storage persistence
- Multiple startup management
- Export/Import functionality

## Files

- `index.html` - Main application file

## Usage

Open `index.html` in a web browser. Add founders, assign tasks and weights, and calculate equity distribution.

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- Local Storage API

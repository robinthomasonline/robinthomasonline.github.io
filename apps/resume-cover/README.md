# Resume & Cover Letter Generator

AI-powered ATS-friendly resume and cover letter generator optimized for international job markets.

## Features

- AI-powered resume generation
- ATS-optimized formatting
- Keyword extraction from job descriptions
- International standards support
- PDF export functionality
- Editable generated content
- Cover letter generation

## Files

- `index.html` - Main application file

## Usage

Open `index.html` in a web browser. Enter your details and job description to generate ATS-friendly resumes.

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- Gemini AI API integration

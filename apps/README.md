# Web Applications

A collection of free web applications and tools built with HTML, CSS, and JavaScript.

## Available Apps

### Productivity & Time Management
- **Work Timer** - Productivity timer with Pomodoro technique
- **Habit Tracker** - Daily habit tracking with streak counters
- **Kanban Board** - Drag-and-drop task management board

### Developer Tools
- **Password Generator** - Secure password generator with compliance presets
- **Subnet Calculator** - Network subnet calculations with binary visualization
- **IP Address Locator** - IP geolocation and network information
- **JSON Formatter** - Format, validate, and minify JSON
- **Markdown Converter** - Convert Markdown to DOC and PDF
- **Image to Favicon Converter** - Generate favicons from images

### Personal & Business
- **Mood Logger** - Track emotional wellbeing and mood patterns
- **Daybook** - Income and expense tracking for small shops
- **Co-Founder Pie** - Calculate equity distribution for startup co-founders
- **Resume & Cover Letter Generator** - AI-powered ATS-friendly resume generator

### Utilities
- **Kerala Lottery Simulator** - Lottery number simulator
- **MyRTA Booking System Monitor** - Browser extension for monitoring booking availability

## Features

- All apps are free to use
- No registration required
- Works offline (where applicable)
- Responsive design
- Local storage support
- Export/Import functionality

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- Local Storage API
- Canvas API (for some apps)
- Service Workers (for PWA support)

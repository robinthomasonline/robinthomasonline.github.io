# Subnet Calculator

Calculate subnet mask, network address, broadcast address, and host range from IP/CIDR notation.

## Features

- IP/CIDR notation support
- Subnet mask calculation
- Network and broadcast addresses
- Host range calculation
- Binary representation visualization
- Network engineering calculations

## Files

- `index.html` - Main application file
- `subnet.css` - Stylesheet
- `subnet.js` - Application logic

## Usage

Enter an IP address with CIDR notation (e.g., 192.168.1.0/24) and the calculator will display all subnet information.

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript

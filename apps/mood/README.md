# Mood Logger

Track your emotional wellbeing and mood patterns over time with interactive charts and analytics.

## Features

- Emoji-based mood tracking
- Daily mood logging with notes
- Interactive charts and analytics
- Mood trend visualization
- Export data to CSV
- Mobile-friendly design
- Local storage persistence

## Files

- `index.html` - Main application file
- `mood.css` - Stylesheet
- `mood.js` - Application logic
- `mood.png` - App icon

## Usage

Open `index.html` in a web browser. All data is stored locally in your browser.

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- Chart.js (for visualizations)
- Local Storage API

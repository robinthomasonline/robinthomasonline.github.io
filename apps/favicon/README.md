# Image to Favicon Converter

Convert any image to favicon format with multiple sizes and preview options.

## Features

- Multiple favicon sizes (16x16, 32x32, 48x48, etc.)
- Image upload and preview
- Individual and bulk download
- Real-time preview
- Supports multiple image formats
- Drag and drop support

## Files

- `index.html` - Main application file
- `favicon.css` - Stylesheet
- `favicon.js` - Application logic

## Usage

Upload an image file and generate favicons in various sizes. Download individually or all at once.

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- Canvas API
- File API

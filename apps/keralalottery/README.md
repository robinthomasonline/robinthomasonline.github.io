# Kerala Lottery Simulator

A lottery number simulator for generating random lottery numbers.

## Features

- Generate random lottery numbers
- Multiple number formats (3-digit, 4-digit)
- Grid-based number display
- Responsible gaming banner
- Simulation disclaimer

## Files

- `index.html` - Main application file
- `style.css` - Stylesheet
- `script.js` - Application logic

## Usage

Open `index.html` in a web browser. Generate random lottery numbers for simulation purposes.

## Disclaimer

This is a simulation tool only. There is no guarantee for choosing lottery numbers using this simulator. The developer has no responsibilities regarding lottery outcomes.

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript

# Assets

Static assets directory containing images, icons, and favicons for the portfolio website and applications.

## Contents

### Application Icons
- App icons and logos for various web applications
- PNG format images for app showcase

### Favicons
- ICO format favicons for different applications
- Multiple sizes for various use cases

### Branding Assets
- Logo images
- Hero images
- Profile images

## File Formats

- PNG - Application icons and images
- ICO - Favicon files
- JPEG - Profile and hero images

## Usage

All assets are referenced using full URLs from the GitHub Pages domain for consistency across the site.
